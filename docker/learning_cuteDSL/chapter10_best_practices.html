Performance Tips

1. Memory Access

  • Use 128-bit vectorized loads/stores
  • Align data to 16-byte boundaries
  • Leverage TMA for global memory access (SM90+)
  • Use multicast to reduce L2 traffic

2. Computation

  • Use Tensor Core operations
  • Keep MMA busy - overlap with memory
  • Use appropriate tile sizes
  • Persistent kernels for large problems

3. Pipeline

  • Use multiple stages to hide latency
  • Balance compute and memory bandwidth
  • Use async copy for shared memory

Common Pitfalls

  • Incorrect strides: Always verify tensor layouts
  • Out-of-bounds access: Add bounds checking for edge cases
  • Missing sync: Ensure proper synchronization before using data
  • Wrong dtype: Match MMA atom with data types
  • Register pressure: Too many register can reduce occupancy
