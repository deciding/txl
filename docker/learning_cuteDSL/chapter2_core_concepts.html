Tensor

A Tensor in CuTeDSL represents a multi-dimensional array with an associated layout. Unlike PyTorch tensors, CuTeDSL tensors are symbolic and can represent slices, partitions, and transformations.

Creating Tensors

import cutlass.cute as cute
import torch

# From PyTorch tensor
torch_tensor = torch.randn(128, 64, device='cuda')
tensor = cute.make_tensor(torch_tensor)

# Create a fragment (register tensor) from tiled MMA
fragment = tiled_mma.make_fragment_A(smem_tensor)

# Create register tensor
reg_tensor = cute.make_rmem_tensor(shape, dtype)

Layout

A Layout describes how multi-dimensional data is mapped to linear memory. It consists of a shape and strides.

Creating Layouts

# Row-major layout (stride[1] = 1)
row_layout = cute.make_layout((128, 64))  # Shape (M, N), stride = (64, 1)

# Column-major layout (stride[0] = 1)
col_layout = cute.make_layout((128, 64), stride=(1, 128))

# From existing tensor
layout = cute.make_layout(torch_tensor)

Key Layout Types

1. ComposedLayout

Nested layouts with outer/inner composition for swizzling:

composed = cute.make_composed_layout(
    outer=cute.make_layout((8, 128)),  # Outer layout
    inner=cute.make_layout((8, 16))       # Inner layout (swizzle)
)

2. Tiled Layout

For dividing work among threads:

# Divide M dimension into 64-element tiles
tiled_layout = cute.tiled_product(
    cute.make_layout((64, 128)),
    shape=(1, 128)  # Tile shape
)

3. Partitioned Layout

For distributing work across threads/warps:

# Partition tensor for threads
partitioned = tiled_mma.partition_A(global_tensor)

Tensor Operations

Slicing

# Slice a dimension
sliced = cute.slice_(tensor, (None, 0))  # Take first row

# Slice to specific range
sliced = cute.slice_(tensor, (slice(0, 64), None))

Local Tile

# Extract tiles from tensor
tiles = cute.local_tile(
    tensor, 
    tile_shape=(32, 32), 
    shift=(0, 0)  # No shift
)
# Result: (num_tiles_m, num_tiles_n, 32, 32)

Local Partition

# Partition tensor among threads
thread_partition = tiled_mma.get_slice(thread_id)
partitioned = thread_partition.partition_A(tensor)

Memory Spaces

Space Abstraction Usage
GMEM Global memory tensor Input/output tensors
SMEM Shared memory tensor Temporary storage, caching
RMEM Register tensor Computation, fragments
TMEM Tensor memory (Hopper+) Large accumulator storage
