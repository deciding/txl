Overview

FMHA (Fused Multi-Head Attention) fuses Q*K^T, softmax, and P*V into a single kernel. The example files are:

  • fmha.py - Forward pass
  • fmha_bwd.py - Backward pass

FMHA Algorithm

# FMHA = Softmax(Q * K^T) * V
# 
# 1. Q*K^T -> S (scores)
# 2. Softmax(S) -> P (probabilities)  
# 3. P*V -> O (output)

Key Features

  • Warp specialization (different warps for different phases)
  • Pipeline stages for overlapping computation
  • Support for causal masking
  • Persistent scheduling for long sequences

Warp Specialization

class BlackwellFusedMultiHeadAttentionForward:
    def __init__(self, ...):
        # Different warps handle different phases
        self.softmax0_warp_ids = (0, 1, 2, 3)  # First softmax phase
        self.softmax1_warp_ids = (4, 5, 6, 7)   # Second softmax phase
        self.correction_warp_ids = (8, 9, 10, 11)  # Bias correction
        self.mma_warp_id = 12                    # Q*K MMA
        self.load_warp_id = 13                   # Load K,V
        self.epilogue_warp_id = 14              # Store output
        self.empty_warp_id = 15                  # Reserved

Tensor Memory (TMEM) Usage

Blackwell introduces Tensor Memory (TMEM) for storing large accumulators 
# TMEM offsets for different buffers
self.tmem_s0_offset = 0    # Score buffer 0
self.tmem_s1_offset = 128  # Score buffer 1
self.tmem_o0_offset = 256 # Output buffer 0
self.tmem_o1_offset = 384  # Output buffer 1
self.tmem_p0_offset = 32   # Probability buffer 0
self.tmem_p1_offset = 160  # Probability buffer 1

Running FMHA

fmha = BlackwellFusedMultiHeadAttentionForward(
    qk_acc_dtype=cutlass.Float32,  # Q*K accumulator type
    pv_acc_dtype=cutlass.Float32,  # P*V accumulator type
    mma_tiler=(128, 128, 64),     # MMA tile shape
    is_persistent=True,             # Persistent kernel
    mask_type=fmha_utils.MaskEnum.NO_MASK
)

fmha(q_ptr, k_ptr, v_ptr, o_ptr, problem_size, ...)
Example command:
python fmha.py --qk_acc_dtype Float32 --pv_acc_dtype Float32 --mma_tiler_mn 128,128 --q_shape 4,1024,8,64 --is_persistent
