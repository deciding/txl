RMSNorm Formula

y = x / sqrt(mean(x²) + eps) * weight

# Equivalent to LayerNorm without mean
# 1. Compute RMS = sqrt(mean(x²) + eps)
# 2. Normalize: y = x / RMS * weight

Cluster-Based Reduction

For large hidden dimensions, multiple CTAs cooperate using clusters:

N ≤ 16K1 (single CTA)
N ≤ 32K2

Configuration Class

class RMSNormConfig:
    def __init__(self, dtype, N, has_weight=True, sm_version=None):
        self.dtype = dtype
        self.N = N
        self.cluster_n = self._compute_cluster_n(N, dtype, sm_version)
        self.N_per_cta = N // self.cluster_n
        self.threads_per_row = self._compute_threads_per_row(N_per_cta)
