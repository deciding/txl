Overview

Grouped GEMM computes multiple independent GEMM operations with different shapes. Files:

  • grouped_gemm.py - Basic grouped GEMM
  • grouped_blockscaled_gemm.py - Block-scaled quantization

When to Use Grouped GEMM

# Traditional: Process each matrix separately
for i in range(num_matrices):
    C[i] = A[i] @ B[i]

# Grouped GEMM: Single kernel for all
# Each CTA handles one matrix, or CTA cooperates within a group

Problem Size Structure

# Each group has its own M, N, K
problem_sizes = [
    (M1, N1, K1),  # Group 0
    (M2, N2, K2),  # Group 1
    (M3, N3, K3),  # Group 2
    ...
]

# Need metadata for each group (offsets, sizes)
cu_seqlens = [0, M1, M1+M2, M1+M2+M3, ...]
