1. Overview

Blockscaled GEMM adds per-block scaling factors for activation quantization.

  • Block Scales: Per-tile scaling factors
  • Mixed Precision: INT8 input, FP16 output
  • Scale Tensor: Separate tensor for scales

2. Scale Tensor Layout

scale_m_tiler = cta_tile_m // block_size
scale_layout = cute.make_layout((scale_m_tiler, num_ctas_n, batch))

Scale Layout

Scales stored per block in M dimension, matching CTA tile distribution.

3. Scale Loading

scale_atom = cute.make_copy_atom(...)
tScaleS, tScaleG = cpasync.tma_partition(...)
tScaleR = cute.make_fragment_like(tScaleS)

cute.copy(scale_atom, tScaleG, tScaleR)

Scale Loading

Scales loaded via TMA alongside main operands, partitioned per thread.

4. MMA with Scaling

cute.gemm(tiled_mma, tCtAcc, tCrA, tCrB, tCtAcc, scale=tScaleR)

Scaled MMA

tcgen05.mma accepts scale parameter to multiply accumulator by block scale.
