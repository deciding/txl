1. Overview

Software pipeline version uses explicit pipelining between warps instead of hardware pipelines.

  • Manual Pipelining: Software-controlled stages
  • Double Buffer: Ping-pong between buffers
  • Warp Sync: Explicit synchronization

2. Pipeline Stages

NUM_STAGES = 3

sA = [smem_a0, smem_a1, smem_a2]  # 3 stage buffers
sB = [smem_b0, smem_b1, smem_b2]

Double Buffering

Multiple buffers allow overlapping load and compute.

3. Software Pipeline Loop

for stage in range(NUM_STAGES):
    current = stage % NUM_STAGES
    next_stage = (stage + 1) % NUM_STAGES
    
    load_to_smem(current)
    compute_from_smem(next_stage)
    sync()

Pipeline Pattern

Load into one buffer while computing from another.
