1. Overview

FMHA backward pass computes gradients for Q, K, V inputs.

  • 3 Gradients: dQ, dK, dV
  • Chain Rule: dP = dO * V, dV = P^T * dO
  • Softmax Grad: Requires sum of dP * P

2. Gradient Computation Flow

1. dO * V -> dP (attention gradient)
2. softmax(dP) -> compute dS (softmax gradient)
3. dS * K -> dQ_grad
4. dS^T * Q -> dK_grad  
5. P^T * dO -> dV

Backward Pass

Similar to forward but with gradient tensors and chain rule operations.
