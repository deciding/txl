Functionality

Initializes FMHA kernel configuration with warp specialization. 16 warps total: - Warps 0-3: Softmax0 - Warps 4-7: Softmax1 - Warps 8-11: Correction - Warp 12: MMA - Warp 13: Load - Warp 14: Epilogue - Warp 15: Empty/auxiliary

TMEM Offsets

  • S0: 0-127, S1: 128-255
  • O0: 256-383, O1: 384-511
  • P0: 32-159, P1: 160-287

        kv_offset = 0 if cum_seqlen_k is None else -s_k * d * h_k

    def correction_rescale(
        self,

Key Takeaways

  1. Warp Specialization: 16 warps with dedicated roles
  2. Pipeline Orchestration: Multiple async pipelines synchronize warps
  3. TMEM Dual-Buffering: S0/S1, O0/O1 for overlapped compute
  4. Numerically Stable Softmax: Subtracts row max before exp
  5. Attention Masking: Supports causal and sliding window
  6. LSE Calculation: Optional log-sum-exp for backward pass
