2. Kernel Class Constructor

class GroupedGemmKernel:
    def __init__(self, acc_dtype, use_2cta_instrs, mma_tiler_mn, cluster_shape_mn, tensormap_update_mode=utils.TensorMapUpdateMode.SMEM):
        self.acc_dtype = acc_dtype
        self.use_2cta_instrs = use_2cta_instrs
        self.cluster_shape_mn = cluster_shape_mn
        self.mma_tiler = (*mma_tiler_mn, 1)
        self.cta_group = tcgen05.CtaGroup.TWO if use_2cta_instrs else tcgen05.CtaGroup.ONE

        self.tensormap_update_mode = tensormap_update_mode
        self.delegate_tensormap_ab_init = (tensormap_update_mode == utils.TensorMapUpdateMode.SMEM)

Functionality

Similar to dense GEMM, plus tensormap update mode (GMEM vs SMEM).

7. Global Offset Calculation

m_offset = cta_id_in_group // num_ctas_n * cta_tile_m

ptr_a = group_base_a + m_offset * k
ptr_b = group_base_b + n_offset

Functionality

Each CTA computes its offset into its group's matrices.

Key Takeaways

  1. Variable sizes: Each group can have different M,N,K
  2. CTA per group: Large groups get multiple CTAs
  3. Continuous storage: All groups stored contiguously
  4. Tensormaps: Can update in GMEM or SMEM
