1. Overview

Demonstrates dependent kernel launches where kernel B's launch depends on kernel A's results.

  • Dependent Kernels: Second kernel needs first's output
  • Launch from Device: Device-side kernel launch
  • CUDA Graphs: Programmatic launch with dependencies

2. Kernel Dependencies

kernel_a(args...)  # First kernel
kernel_b(args...)  # Depends on kernel_a

with cuda.graph():
    node_a = cuda.graph.add_kernel(kernel_a)
    node_b = cuda.graph.add_kernel(kernel_b)
    cuda.graph.add_dependency(node_b, node_a)
