TeraXLang IR Viewer Help

Overview

This tool shows the binding between IR code and Python source. Drag & drop files or click to browse, then click "Load & View".

Upload

  • Drag & drop files into the drop zones
  • Click drop zones to browse for files
  • Or use the original file input method

Navigation

  • Click IR line → Highlights bound Python lines
  • Click Python line → Highlights bound IR lines

Supported Formats

  • .ttir - Triton TTIR
  • .ttgir - Triton TTGIR
  • .ptx - NVIDIA PTX
